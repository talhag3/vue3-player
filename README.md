# Vue 3 Music Player

A simple and responsive music player built with Vue 3, Pinia, and Tailwind CSS. The app features a clean UI, playback controls, and a predefined playlist.

## Features

- **Play/Pause/Seek**: Control playback with buttons and a slider.
- **Next/Previous Track**: Navigate through the playlist.
- **Track List**: Click on any track to start playback.
- **SVG Icons**: Custom icons for a lightweight and modern design.
- **Error Handling**: Display error messages for invalid operations.
- **Real-Time Progress**: Visual feedback on playback progress.

## Technologies Used

- **Vue 3** with `<script setup>` and Composition API
- **Pinia** for state management
- **Tailwind CSS** for styling

## Installation and Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/vue3-music-player.git
   cd vue3-music-player

```

2.  **Install Dependencies**
    
    ```bash
    npm install
    
    ```
    
3.  **Run the Development Server**
    
    ```bash
    npm run dev
    
    ```
    
4.  Open your browser at [http://localhost:3000](http://localhost:3000/).
    

## Usage

-   Click on the play button to start playback.
-   Use the progress slider to seek within a track.
-   Navigate through tracks using the next/previous buttons or by selecting a track from the list.
